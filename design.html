DESIGN DU SITE

Cette page vous décrira le public cible et vous expliquera pourquoi le site web suit les bonnes pratiques de conception web en termes de couleurs et configuration texte, par exemple, parmi d'autres détails.

Public cible

Notre site web a été conçu pour les gens qui aiment expérienter un restaurant avec l’ambiance luxueuse. En ce sens, nous avons cherché:

  • Utiliser des images qui attirent l'attention du public.
  • Utiliser des couleurs basé sur noir et rouge qui correspondent la décoration de notre restaurant.
  • Organiser le site de manière hiérarchique.
  • Présenter les informations clés dans le gauche de siteweb pour facilitation de les trouver par les clients.

Respect de bonnes pratiques du design web

Répétition

  • Les mêmes couleurs sont utilisées sur toutes les pages.
  • Les entêtes h1, h2 et h2 sont identiques dans toutes les pages.
  • La même police a été utilisée sur toutes les pages.
  • L'en-tête, la navigation et le pied de page sont identiques sur chaque page.

Contraste

  • Le bon contraste entre le texte et l'arrière-plan a été confirmé sur le site web WebAIM.
  • Les images sélectionnées se distinguent également par rapport à l'arrière-plan.

Proximité

  • Le menu de navigation regroupe les liens dans le même contexte.

Alignement

  • Le contenu de la page sont séparer par différents divisions: l'information clé, navigation et le main contenue.
  • Les titres des pages sont toujours alignés en haut à gauche.
  • Les texts sont présentés par différents techniques selon le contenue: pour le menu, on les présente en tableaux tandis que le secteur réservation on utilise formulaire.
  • Les images sont bien choisis pour s'adapter le text, et tirent le désir de nous visiter.

Présentation de l'information

  • Le titre de chaque page est toujours affiché à travers l'en-tête h1 et positionné juste en dessous du menu de navigation, ce qui facilite la localisation de l'utilisateur.
  • Les polices sans-serif et les tailles choisis sont faciles à lire.
  • Les paragraphes sont courts.
  • Le contenu du site s'ajuste à la taille de l'écran du navigateur, en respectant une taille prédéfinie minimale ou maximale.
  • Notre siteweb s'adapte par différents mesures: ordinateur, tablette et cèlulaire intelligent.

Respect des critères d'accessibilité

Percevable

  • Alternative textuelle dans les images (toutes les images ont été marquées à l'aide de l'attribut alt).

Opérable

  • Tout peut être fait avec le clavier.
  • Il n'y a pas de contenu multimédia clignotement (flashing) qui peut causer une crise (seizure).

Compréhensible

  • Le site est très simple et facile à comprendre.

Robuste

  • Le site a été testé et fonctionne sur Firefox, Chrome et Safari.

Effets visuels

Images

  • Les résolutions des images sont de bonne qualité et les tailles des fichiers sont raisonnables, ce qui assure la rapidité d'affichage du site web.

Couleurs

Les couleurs du site ont été choisies basé sur les couleurs qui correspondent la décoration de notre restaurant et le logo de notre marque.
